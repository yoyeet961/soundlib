# Soundlib for Python
A bunch of sounds you can instantly make.

# Usage
How to make a beep sound:

`soundlib.beep()`

How to make a victory sound:

`soundlib.win()`

Using Visual Studio Code and getting an error? Use these!:

`soundlib.vwin()`
`soundlib.vbeep()`
